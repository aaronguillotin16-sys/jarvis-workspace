# Livrables

Ce dossier contient tout ce que Claude produit pour Aaron. C'est la sortie du Jarvis.

---

## Règle d'or : Inputs / Outputs

| Quoi | Où ça va |
|------|----------|
| Documents que tu fournis à Claude (PDFs, notes, exports, screenshots) | `context/import/` |
| Ce que Claude produit pour toi (sites, flyers, logos, diapos...) | `livrables/` |

---

## Structure des sous-dossiers

| Dossier | Contenu |
|---------|---------|
| `sites/` | Sites web complets ou pages créées pour des clients ou projets perso |
| `flyers/` | Flyers, affiches, visuels promotionnels |
| `diapos/` | Présentations et diaporamas |
| `logos/` | Logos et identités visuelles |
| `videos/` | Scripts, storyboards, contenus vidéo |
| `apps/` | Applications web ou mobiles |

---

## Convention de nommage

Format : `YYYY-MM-DD_nom-du-projet`

- Tout en minuscules
- Mots séparés par des tirets
- Date au début pour trier chronologiquement
- Nom court mais explicite

**Exemples :**
```
2026-05-28_restaurant-le-port
2026-06-01_boutique-sneakers
2026-06-15_flyer-promo-ete
```

Si un projet a plusieurs fichiers, crée un sous-dossier avec ce nom et range tout dedans.
