## Le projet

Site vitrine mobile-first pour un snack local. Objectif : montrer le prototype au gérant et décrocher la mission.

**Ce que le site fait :**
- Hero scroll-expansion avec animation burger
- Carte complète : Sandwichs, Burgers, Tacos, Assiettes (23 produits)
- Style dark industriel avec watermark BUTCHER, badges étoile dorés
- 2 colonnes sur mobile, tap sur une carte = description
- Section commande : bouton d'appel → confirmation → barre de progression Uber Eats style

---

## Stack technique

- Next.js 16 + TypeScript + Tailwind CSS v4
- Framer Motion (animations)
- react-parallax-tilt (effet 3D cartes)
